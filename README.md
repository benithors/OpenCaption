<h1 align="center">OpenCaption</h1>

<p align="center">
  A macOS desktop app for burning subtitles into social videos.
</p>

---

## Features

- Drop in a video, get captions burned in — ready for TikTok, Reels, Shorts
- OpenAI Whisper transcription (or bring your own transcript)
- Style presets for social platforms
- Built with Electron + Remotion + Vite

## Getting Started

```bash
npm install
npm run dev
```

## Export

The app renders your captioned video using Remotion and FFmpeg — no cloud needed.
