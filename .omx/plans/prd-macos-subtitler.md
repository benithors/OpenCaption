# PRD Outline — macOS Subtitler

## User
macOS creator / social video editor.

## Problem
Turn spoken video into styled burned-in subtitles quickly.

## Core Flow
Import video → provide API key → transcribe → correct text → style → export.

## In Scope
- OpenAI transcription (`gpt-4o-transcribe` / `gpt-4o-mini-transcribe`)
- Subtitle preview/editing
- Styling controls and presets
- Local burned-in export via Remotion

## Out of Scope
- Full timeline editing
- Subtitle timing editing
- Multiple transcription providers

## Success Criteria
A representative user can complete the flow in one session on sample clips.
